# My Dotfiles

It includes the config for:
- `atuin` : terminal history management
- `bat` : better `cat`
- `thefuck` : auto corrected command
- `git` : version control
- `oh my zsh` : zsh framework
- `powerlevel10k` : zsh shell theme
- `wezterm` : terminal emulator

## Setup for new mac
- sh setup_mac.sh

## Setup git
- sh setup_git.sh
